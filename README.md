## Installation

To get started with this project, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/HuuKienIT/chair-configurator.git
   cd chair-configurator
   ```

2. Install the dependencies using Yarn:

   ```bash
   yarn install
   ```

## Usage
